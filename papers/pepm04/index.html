Program codes and an online appendix for "Fusion with Stacks and Accumulating Parameters"

This directory contains some contents that could not be included in the paper "Fusion with Stacks and Accumulating Parameters", which is authored by Susumu Nishimura and is to appear in PEPM'04 proceedings.

Sample program codes

A full set of program codes of the running example is available as a tar+gzip'd archive. Every program contained in this archive is executable in Hugs.

Online appendix

The proof of theorems is available as an online appendix (gzip'd postscript).
